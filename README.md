# LAvatar

<p>이 어플리케이션은 PC 게임 LostArk 플레이어들의 아이템 거래를 돕고자 제작되었습니다. 지금까지 LostArk 에서 발매한 아바타세트를 총 망라하였으며, 해당 세트의 구성품의 가격 정보를 한 눈에 확인할 수 있도록 정리하였습니다. 아이템 가격 정보는 1시간 주기로 갱신됩니다.</p>
